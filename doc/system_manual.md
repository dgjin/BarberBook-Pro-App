
# BarberBook Pro 系统操作手册

**版本**: 1.5.6  
**更新日期**: 2024-03-28

---

## 1. 顾客端操作指南

### 1.1 注册与登录
- **新用户**: 在启动页选择“我是顾客” -> “立即注册”，上传头像并设置密码。
- **老用户**: 输入手机号与密码登录。初始演示账号为 `13888888888 / 123456`。

### 1.2 预约服务
- **专家预约**: 首页选择心仪理发师，进入详情页查看其擅长领域与评价。
- **时段锁定**: 挑选日期与时间段。灰色时段代表已约满、已过期或理发师休假。
- **凭证生成**: 确认预约后，系统生成专属预约码，保存在“个人中心”。

### 1.3 签到与排队
- **到店激活**: 到店后向理发师展示预约码。
- **实时看板**: 通过“监控大屏”或个人中心查看预计等待时长及候补位次。

## 2. 理发师工作台指引 (重点更新)

### 2.1 实时叫号与接待
- **今日队列**: “今日队列”页签展示按时间排序的所有预约。
- **签到激活**: 
  - **扫码签到**: 点击“扫描签到”调用相机扫描顾客二维码。
  - **手动补登**: 若相机无法识别，可向顾客询问其预约 ID（纯数字），点击扫描页面的手动输入完成签到。
- **开始服务**: 签到后的顾客会出现在“正在接待中”区域。

### 2.2 结算与统计对账
- **一键结算**: 服务完成后点击“确认完成结算”，系统将自动触发财务对账流程。
- **数据看板**:
  - **今日完结订单**: 实时统计今日状态为 `completed` 的单量。
  - **今日流水营收**: 实时累计今日已结算订单的总金额。
  - **职业成就**: 包含“服务人次总计”与“券额营收总计”，数据直接溯源至历史全量流水，确保荣誉真实性。

## 3. 管理员后台指引

### 3.1 财务数据中心
- **年度看板**: 切换至年度视图，查看全店及各发型师的理发券核销贡献度。
- **实时校验**: 所有数据支持点击“同步”按钮进行云端实时对账，消除脏数据影响。

### 3.2 系统级配置
- **运营参数**: 调整营业时间后，前端预约日历会即时根据新时段和预设的服务时长（默认45min）重绘。
- **审计记录**: 在“日志中心”可追溯任何由于取消服务导致的财务冲正记录。

## 4. 常见问题 (FAQ)

**Q: 为什么工作台统计的“今日营收”比我实际做的人头数少？**  
A: 系统仅统计已点击“确认完成结算”的订单。请确保每个服务项目都执行了结算操作。

**Q: 生涯成就的数据会丢失吗？**  
A: 不会。生涯数据基于 `app_appointments` 的历史全量记录实时聚合生成，只要数据库记录存在，成就永远有效且实时更新。

**Q: 顾客取消预约后，我的业绩会变吗？**  
A: 若订单已结算后被管理员取消，系统会自动执行“财务冲正”，从统计中扣减对应金额，确保财务报表准确。
