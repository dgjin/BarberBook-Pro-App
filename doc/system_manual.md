
# BarberBook Pro 系统操作手册

**版本**: 1.4.2  

---

## 1. 顾客端操作指南

### 1.1 注册与登录
- **新用户**: 在启动页选择“我是顾客” -> “立即注册”，上传头像并设置密码。
- **老用户**: 输入手机号与密码登录。初始演示账号为 `13888888888 / 123456`。

### 1.2 预约服务
- 首页选择心仪理发师。
- 挑选日期与时间段。注意：灰色时段代表已约满或已过期。
- 确认支付后，预约凭证将保存在“个人中心”。

### 1.3 签到与排队
- 到店后，在个人中心点击“立即签到”或向理发师展示预约二维码。
- 实时通过“监控大屏”查看自己前面还有多少人。

---

## 2. 理发师工作台指引

### 2.1 客户接待
- 扫码签到：点击工作台“扫描客户签到”按钮，扫描顾客二维码。
- 开始服务：客户状态变为“已到店”后，即可开始理发。

### 2.2 完工与收益
- 点击“完成服务”，系统将自动检测客户理发券并执行核销。
- 成功核销后，您的个人业绩将实时增加。

---

## 3. 管理员后台指引

### 3.1 财务看板 (重点更新)
- 点击底部导航“看板”。
- 点击右上角**钱包图标**切换至年度看板。
- 您可以查看到当前年份全店的总核销张数及每位发型师的业绩贡献排行。

### 3.2 人员与系统管理
- **人员**: 修改理发师职级、头像，或者为顾客手动补发理发券。
- **日志**: 查看所有预约撤销、系统更新等关键操作记录。
- **设置**: 调整店内营业时间、新增理发套餐。

## 4. 常见问题 (FAQ)

**Q: 为什么我帮客户手动取消预约后，理发师的钱（张数）变少了？**  
A: 这是系统的自动财务纠错机制。若该订单之前已计入理发师业绩，取消时会同步冲正，确保财务对账准确。

**Q: 如何开启大屏显示？**  
A: 在监控页面点击“大屏模式”，系统将开启语音叫号与专业监控视图。建议在店内电视或 Pad 上展示。
